## 🇨🇳 中国大陆

### ⚠️ 总体情况说明

中国大陆的博物馆数字化投入巨大（故宫、敦煌、国博等都有非常出色的数字化成果），但**开放策略与欧美/日本有本质区别**：

- **不提供标准开发者 API**（无 REST / GraphQL / IIIF 公开接口）
- **不采用 CC0 或 Creative Commons 授权**（文物数字资源版权归博物馆所有）
- **浏览免费，但下载/商用需走授权审批**
- 2025年国家发布了《博物馆数字资源许可审查操作指南》，商用授权通道正在**逐步规范化**

## 💡 特别说明：中国大陆的现状与趋势

### 为什么中国大陆博物馆不提供 API？

1. **版权观念不同**：中国大陆法律体系中，文物数字影像被视为博物馆的"数字资源资产"，不采用 CC0/CC BY 等放弃版权的方式
2. **政策在变化**：2025年底《博物馆数字资源许可审查操作指南》发布，正在建立规范化的授权体系
3. **商业模式探索中**：故宫文创、数字敦煌等已有成熟的商用授权路径
4. **内部渠道可能更高效**：如果公司与故宫/敦煌/国博等已有合作关系，走内部渠道比公开 API 更快

### 建议策略

对于中国大陆资源：
- **短期**：通过数字敦煌的分级授权体系申请商用素材
- **中期**：关注《博物馆数字资源许可审查操作指南》落地后的新开放政策
- **长期**：探索与故宫/国博等建立正式的数字资源合作协议
