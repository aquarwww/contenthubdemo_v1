## A. 武器与甲胄专题

对游戏项目的武器系统、角色装备设计有**直接参考价值**。

### 23. Royal Armouries（英国皇家军械博物馆）🇬🇧

| 项 | 内容 |
|---|------|
| **官网** | https://royalarmouries.org |
| **在线藏品** | https://collections.royalarmouries.org |
| **规模** | **7.5万+** 件武器与盔甲（从中世纪到现代） |
| **API** | ⚠️ 无标准 REST API，有在线检索系统可结构化查询 |
| **授权** | 在线浏览免费，商用图片需联系授权 |
| **内容亮点** | 全球最大的武器与盔甲收藏之一——亨利八世盔甲、印度战象甲、日本武士刀、东方武器 |
| **游戏价值** | ⭐⭐⭐⭐⭐ 武器/盔甲设计的终极参考库 |
| **接入方式** | 在线检索 + 人工获取高清图 |
